# URL to fetch and Filter 'uFilter' :fire:

#Install & Set :
```
> git clone https://github.com/RootKRD/Filter
> cd Filter
> python uFilter.py
```

# Usage : 

```
python uFilter.py -h
python uFilter.py https://example.com -txt
python uFilter.py https://example.com -txt -js -json
```
